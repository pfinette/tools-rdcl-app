Adding more tools

This page is intentionally simple so you can drop in more single‑file tools as you create them.

  1. Create a new tool file. Put your HTML tool in the same folder as this landing page. Example: baseline-grid.html
  2. Duplicate a card. In this file, copy one <a class="tool-card" ...> block inside the <div id="tools"> container.
  3. Update its details. Change the href, titles, description, and the small filename label so they match your new tool.

Tip: Because everything is plain HTML, you can open the landing page and each tool directly from your file system without any server.
